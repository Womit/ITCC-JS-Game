About the Game

This game is a simple hopper game. The aim of the game is to score 20 points by jumping over enemies. If you fail and get hit by an enemy you lose and have to restart :(

The inspiration behind this game is from the Google Chrome Dinosaur Hopper game. I thought it would be a fun game to use as a baseline and change some of the rules and the art style too. I thought it would be fun to use my own sprites for the character and enemy in the game along with the background. The game from Chrome was very quiet every time I played it so I decided to add my own sounds to my version.
